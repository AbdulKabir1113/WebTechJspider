Array Method
----------------
push()
-------
* The push method is used to add the element at the end of the array (or) at the tail of the array
* It will accept n number of arguments.
ex:
var arr = [10,20]
arr.push(40,30)
console.log(arr);//[10,20,40,30]


unshift()
-------
* The unshift method is used to add the element at the starting of the array (or) at the head of  array
* It will accept n number of arguments.
ex:
var arr = [10,20]
arr.unshift(40,30)
console.log(arr);//[40,30,10,20]

pop()
-------
* The pop method is used to remove elements from the end of the array (or) tail of the array
* It accepts no arguments

ex:
var arr = [10,20,40,30]
arr.pop()
console.log(arr);//[10,20,40]

shift()
-------
* The shift method is used to remove elements from the start of the array (or) at the head of a array
* It accepts no arguments

ex:
var arr = [10,20,40,30]
arr.shift()
console.log(arr);//[20,40,30]

indexof()
---------
* the indexof() method it returns the index of particular elemet which is passed through it.
* arr.indexOf(item, from) – looks for item starting from index from, and returns the index where it was found, otherwise -1

var arr = [10,210,30]
arr.indexof(20);//2

map()
-----
* The map() method in JavaScript creates a new array by applying a function to each element of the original array. 
* it is a higher order method
* It skips empty elements and does not alter the original array, making it ideal for transforming data.
* It returns a new array and elements of arrays are the result of the callback function. 

eg:
     let  arr = [10,20,30,40]
    
    let  a1 = arr.map((e)=>{
	return e;
	})

       console.log(a1)

reduce()
---------
* The reduce() method in JavaScript executes a reducer function for each array element, returning a single accumulated value.
* it is a higher order method
* it will return sum of an array elements

eg:
     let  arr = [10,20,30,40]
    
    let  a1 = arr.reduce((acc,value)=>{
	return acc+value;
	})

       console.log(a1)

join()
---------
* The JavaScript Array join() Method is used to join the elements of an array into a string.

------

STRINGS

The JavaScript string is a sequence of characters.
There are 2 ways to create string in JavaScript
1. By string literal
2. By string object (using new keyword)
3.By using constructor way


1) By string literal
The string literal is created using double quotes. 
var stringname="string value"; 

2) By string object (using new keyword)
The syntax of creating string object using new keyword is given below:
var stringname=new String("string literal"); 
Here, new keyword is used to create instance of string.

3) By constructor way (without using new keyword)
The syntax of creating string object without using new keyword is given below:
var stringname=String("string literal"); 
Here, new keyword is not used to create instance of string.

OBJECT

*Any substance which has its existance in the real world is known as OBJECT.
* Every object will have properties(attributes of feeds which describe them).
* Every object will have actions(behaviours)
EX:
1.car:
properties of car: model,color,price
actions of car:accelarate,brake,start,stop
2.webpage:
properties of webpage:url,data/content on webpage.
actions of webpage:scroll,hover,click
Note:
* In programming an object is a block of memory which represents a real world.
* The properties of real world object are represented using variables
* The actions of real world object are represented using METHODS(FUNCTIONS).
* In javascript object is a collection of attributes and actions in the form 
of key&value pairs enclosed within curly braces{}
* Key represents properties/attributes of an object
* Value represents state of an object.
-In javascript,we can create js object in 3 different ways:

1.Object literal{}
------------------
ex:

let obj={
    name:"reshma",
    id:1
}
console.log(obj);

2.Using a function
-------------------
ex:
function obj1(name,id){
    this.name=name;
    this.id=id;
}
let obj2=new obj1('reshma',100)
console.log(obj2);

3.Using a new keyword
----------------------------
ex:

let obj=new Object()
console.log(obj);

4.using a constructor way
---------------------------
ex:
let obj=Object()
console.log(obj);

Note:
* If the key is not present in the object then we get 'undefined' as the value.

* The objects in javascript are mutable(state of the object can be modified) in nature.

Object Methods:

Object.create()
This method is used to create a new object with the specified prototype object and properties.

Object.entries()
This method returns an array with arrays of the key, value pairs.

Object.freeze()
This method prevents existing properties from being removed.(we cannot update new key and value pair and also previous key and value pair)

Object.keys()
This method returns an array of a given object's own property names.

Object.seal()
This method prevents new properties from being added and marks all existing properties as non-configurable.(we cannot update new key and value pair but we can update previous key and value pair)

Object.values()
  This method returns an array of values

Destructuring

* It allows to convert elements into values from arrays, or properties from objects, into distinct variables data can be extracted from arrays, objects, and nested objects, and assigned to variables. 
* In Destructuring Assignment on the left-hand side, we define which value should be unpacked from the sourced variable.

* It allows to convert array elements into values and object keys into variables and this is called destructuring.

let names = ["alpha", "beta", "gamma", "delta"];

let firstName = names[0];
let secondName = names[1];

console.log(firstName);

let arr=[10,30,40]

             name : "abc",

Rest parameters(. . .)

*The rest parameter it allows a function to accept an indefinite number of arguments as an array(packing the data)
*A function definition can only have one rest parameter.
*The rest parameter must be the last parameter in the function definition.

eg: 
    function demo(a,...b){
        console.log(a)
        console.log(b)
    }
demo(10,20,30,40,50)



Spread parameters(. . .)

*Spread parameter looks exactly like rest parameter. In a way, spread parameter is the opposite of rest parameter. (unpacking the data)
*Spread pararmeter "expands" an array into its elements, while rest parameter collects multiple elements and "condenses" them into a single element.

let arr=[10,20,30]
let arr1=[50,80,90]

let c=[...arr,...arr1]
console.log(c)//[10,20,30,50,80,90]
